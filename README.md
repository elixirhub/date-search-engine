# Datich


## How to use

```Javascript
var datich = require('datich');

var result = datich(html);
```


## Result

```Javascript
dates [
  {
    value
    index
    isd // international standard date
  }
]
```


## Command line utilization

```Shell
node datich.js http://elixir-europe.org/events
```


## Installation

Manually:

```Shell
git clone git://github.com/elixirhub/datich.git
```
